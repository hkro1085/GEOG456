California Wildlife Movement Barriers

Inspiration and help with code was taken from the posted example: https://github.com/registerchilton/Geography456/blob/main/Northside/NorthsidePolygons.html and previous HW examples.
Limitations to this data include the fact that there are only 2 years of data available, so it is difficult to assess long-term trends, however an extention of this homeowrk could be pulling more barrier data to add temporal complexity. Overall this Visualization is meant to allow users to see which wildlife movement cooridoors need the most attention.

Key:
Red = Top Priority Barrier
Orange = Priority Barrier
Yellow = Barrier
Green = Remediated
